About the Project

It is a real-time chat application built using the following technologies:

Backend:

  • Spring Boot
  • PostgreSQL
  • WebSocket
  • STOMP
  • Docker

Frontend:

  • Thymeleaf
  • CSS

Chat features:

  • - Basic username&password login
  • - Public & Private messages
  • - Available history of public and private messages
  • - sending emojis
  • - Customizable profile settings: Nickname, password and profile picture
  • - Admin account: admin can kick user from chat room and BAN/UNBAN user, auto BAN expire check
